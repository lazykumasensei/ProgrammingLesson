# Android개발을 위한 프로그래밍 레슨2

## 레슨 목표
- 기존에 작업한 Layout을 이용해서 메인페이지 완성
- Class 기초 
- 컴포넌트 (View) 기초
- Kotlin Extension을 이용한 컴포넌트 연결 및 사용
- 반복되는 디자인을 제어하고 데이터를 연결하는 방법 (GridView)
- 리스트를 구성하기 위한 Adapter
- 페이징 처리를 위한 ViewPager2와 Fragment

## 소요 수업
- 3 ~ 5 수업
